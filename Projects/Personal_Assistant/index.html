Virtant - Personal Assistant

Nowadays passwords have become the essential key to access any web pages or to login any application. An average user accesses numerous sites that it becomes almost impossible for the user to remember all those passwords. This led to problem like password reuse, in which the user uses same password for multiple sites and applications, which create a vulnerability by which the hacker can access the user content easily. There are also cases where users save their important credentials like net-banking’s password without proper security in places like phone book, notes or their personal diaries which can be accessed by any other attacker.
