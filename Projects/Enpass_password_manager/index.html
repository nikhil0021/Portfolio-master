EnPass – The Password Guard

Nowadays passwords have become the essential key to access any web pages or to login any application. An average user accesses numerous sites that it becomes almost impossible for the user to remember all those passwords. This led to problem like password reuse, in which the user uses same password for multiple sites and applications, which create a vulnerability by which the hacker can access the user content easily. There are also cases where users save their important credentials like net-banking’s password without proper security in places like phone book, notes or their personal diaries which can be accessed by any other attacker.

Enpass will ease the things for you by storing the password on its own database and providing them to you whenever needed. Enpass is an android application which can be used by user wherever and whenever user wants to use in which n-numbers of passwords can be retrieved by the user according to its account name and master password. The master password will remain same for all log which is saved by the user. The password which will be stored in each log is saved in encrypted form in the database. AES 256-bit encryption and some hashing technique is implemented in this password manager application for securing user’s credentials.
Technology used: Android, Firebase, AES-256, javamail API
Module Developed: Data Encryption, Online database storage, CRUD operation, User Profile Management, Contact Admin, e-Mail functionality, User Authentication/Validation/Verification.
